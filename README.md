
# Tarea S1.05: Java Utilities

## Introducción
Este proyecto contiene ejercicios en Java enfocados en el manejo de archivos, directorios y estructuras. Cada ejercicio se encuentra en la carpeta `src/nivel_1` y cuenta con un archivo `Main.java` como punto de entrada.

---

## Instrucciones de Uso

### 1. Abrir la consola
Por defecto, la consola estará ubicada en:  
```bash
C:\Users\User>
```

Navega hasta la carpeta `src` usando el comando `cd`:
```bash
cd IdeaProjects\Tarea_S1.05_Java_Utils\src
```

### 2. Compilar el archivo
Compila el archivo que contiene el método `main` con el comando `javac`. Por ejemplo, para el ejercicio 1:

```bash
javac nivel_1/ejercicio1/Main.java nivel_1/ejercicio1/ListDirectory.java
```

### 3. Ejecutar el archivo
Ejecuta el archivo `.class` que se genera, proporcionando los argumentos que requiere. Por ejemplo, para el ejercicio 5:

```bash
java nivel_1.ejercicio5.Main src/nivel_1/ejercicio5
```

Al ejecutar el comando, se mostrará la acción programada en el método `main` del ejercicio correspondiente.

---

## Requisitos
- **Java 8** o superior.
- **IntelliJ IDEA** o cualquier IDE compatible.
- Compatible con **Windows**, **macOS** o **Linux**.
